CSS externo

É muito simples para organização de diversas páginas, pois com um link de estilo, sim, um link de estilo que pode ser usado em todas as páginas do site, irá facilitar muito o aprendizado

Como eu consigo fazer um link externo com CSS?

A resposta é muito simples, dentro da tag head, basta colocar o seguinte link: <link rel="stylesheet" href="style.css">

Este comando irá criar uma página externa onde você poderá manipular os elementos HTML com CSS e fazer um projeto mais organizado e simples de ler.
